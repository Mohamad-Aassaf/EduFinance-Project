# Projeto EduFinance

## Descrição
O **EduFinance** é uma plataforma educacional voltada ao ensino de educação financeira, com foco em facilitar a compreensão de conceitos econômicos por meio de dados reais, gráficos interativos e linguagem acessível.  
O projeto está sendo desenvolvido como Trabalho de Conclusão de Curso (TCC) para o Instituto Federal Sul-riograndense Campus Sapucaia do Sul.

## Objetivo
O objetivo principal do projeto é auxiliar estudantes e iniciantes a compreenderem indicadores financeiros e econômicos, promovendo educação financeira de forma prática e visual.

## Funcionalidades
- Visualização de dados financeiros em tempo real
- Gráficos interativos
- Interface web acessível
- Consumo de APIs financeiras
- Conteúdo educativo voltado para leigos e jovens

## Tecnologias Utilizadas
- HTML, CSS e JavaScript
- API Fetch
- APIs de dados financeiros
- Biblioteca de gráficos (ex: ApexCharts)

## Público-alvo
Pessoas interessadas em aprender educação financeira, incluindo estudantes do ensino médio, superior e público geral.

## Justificativa
A educação financeira é essencial para a tomada de decisões conscientes no cotidiano. O EduFinance surge como uma solução tecnológica para reduzir a complexidade dos dados financeiros e torná-los mais compreensíveis.
